At **Shanghai Jiao Tong University**, I built a session-memory component for a mental-health support agent using LangGraph, FastAPI, and PostgreSQL/pgvector. Drawing on cognitive-architecture theory (CoALA), I decomposed "memory" into three layers—semantic facts, episodic arcs, and procedural rules—ranked by retrieval priority and linked by dependency, turning fragmented conversations into a longitudinal psychological-state chain across five interfaces (CLI, web chat, web voice, Telegram, and a backend API).

**Stack:** LangGraph · FastAPI · pgvector · CoALA
